# SR Verge — Icon & Logo File Map

This is the complete map of every image that makes up the app's branding, so you can
replace them with your own. Files are grouped by what they affect.

---

## 1. App icons (window, taskbar, .exe, installer) — **most important**

Location: `src-tauri/icons/`

| File | Used for |
|---|---|
| `icon.png` | Master source image (square, ideally **1024×1024**) |
| `icon.ico` | Windows app/exe icon |
| `icon.icns` | macOS app icon |
| `32x32.png` | Small icon (tray fallback, small UI) |
| `128x128.png` | Standard app icon |
| `128x128@2x.png` | Retina app icon (256×256) |

These are the ones declared in `src-tauri/tauri.conf.json` → `bundle.icon`.

### Easiest way to replace all of the above at once
Make one square PNG (1024×1024, transparent background) and run:

```bash
pnpm tauri icon path/to/your-icon.png
```

This regenerates `icon.ico`, `icon.icns`, all the PNGs **and** the Windows Store
tiles below — from your single source image. (It does **not** touch the tray icons
or the in-app SVG logo in sections 3–4; replace those by hand.)

## 3. System tray icons

Location: `src-tauri/icons/`

| File | State |
|---|---|
| `tray-icon.ico` | Default tray icon (colored) |
| `tray-icon-mono.ico` | Default, monochrome |
| `tray-icon-sys.ico` | System-proxy mode (colored) |
| `tray-icon-sys-mono.ico` / `tray-icon-sys-mono-new.ico` | System-proxy mode, monochrome |
| `tray-icon-tun.ico` | TUN mode (colored) |
| `tray-icon-tun-mono.ico` / `tray-icon-tun-mono-new.ico` | TUN mode, monochrome |

These show the proxy state in the OS tray. Replace by hand (keep the same filenames).
`-mono` versions are for macOS/Linux menu bars that want a single-color glyph.

## Quick start (recommended order)

1. Design one **1024×1024 PNG** of your logo → run `pnpm tauri icon your-logo.png`
   (covers sections 1 & 2).
2. Replace the **tray icons** in `src-tauri/icons/tray-icon*.ico` (section 3).
3. Replace `src/assets/image/icon_light.svg` + `icon_dark.svg` (header glyph) and
   `logo.ico` (favicon). Edit `logo.svg` if you want a graphic wordmark instead of text.
4. Rebuild: `pnpm build:fast`.
